Ejemplos de APIs de JavaScript

API Web Storage

La API Web Storage permite almacenar datos de forma persistente en el navegador. Se pueden usar dos tipos de almacenamiento: localStorage y sessionStorage.

Su importancia nos sirve evitar la necesidad de utilizar cookies.

API Forms

La API Forms permite interactuar con los formularios HTML y y su importancia incide en validar los datos antes de enviarlos al servidor.





API Web Worker

La API Web Worker permite ejecutar scripts en segundo plano para evitar bloquear la interfaz de usuario.

Su importancia es grande porque al ejecutarse en segundo plano no molestará al usuario

API fetch

La API fetch permite enviar solicitudes HTTP desde el navegador y recibir respuestas.

Su importancia incide en la recepcion de respuestas desde el navegador

API Geolocation

La API Geolocation permite obtener la ubicación del usuario mediante el uso de la información de su dispositivo.

Su importancia es grande porque nos puede servir para localizar ciertas direcciones como la del usuario

API History

La API History proporciona una manera de navegar hacia atrás y adelante en la historia del navegador web.

Su importancia podria ser para un menu y regresar a la pestaña de atras o a la pestaña que se desea
